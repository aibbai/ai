6장 모듈과패키지
1절. 파이썬 표준 모듈 사용
	- 모듈/패키지 or 모듈 안의 함수 가져오는 방법

		import A                    - A 모듈을 가져옴
		import A as B              - A 모듈을 B 이름으로 가져옴
		from A import B           - A 모듈안의 B 함수(클래스|변수)를 가져옴
		from A import B as C    - A 모듈안의 B 함수를 C 이름으로 가져옴

		from A import B           - A 패키지 안의 B 모듈을 가져옴	
		from A.B import C         - A 패키지 안의 B 모듈안의 C 함수 가져옴
		from A import B as C     - A 패키지 안의 B 모듈을 C 이름으로 가져옴
		from A.B import C as D   - A 패키지 안의 B 모듈안의 C 를 D 이름으로 가져옴

2절. 사용자 정의 모듈
	- 함수, 클래스, 변수 등을 파이썬 파일(*.py)에 저장해서 모듈을 만들고, 재사용 가능
	- 모듈을 가져오는 순서 :
			(1) 표준 모듈 
			(2) sys.path 변수에 저장된 경로

3절. 패키지
	- 모듈이 들어있는 폴더에는 반드시 __innit__.py 파일 만들기
	- 상위 패키지 import > 하위 패키지까지 import 안됨
	- 패키지 import > 모듈 import 안됨


4절. 패키지 설치 및 삭제 방법

	아나콘다 프롬프트에서 
		- 패키지 설치 : pip install 패키지명[==버전] / conda install 패키지명[==버전] / uv pip install 패키지명[==버전]
		- 패키지 확인 : pip show 패키지명           / conda list 패키지명
		- 패키지 삭제 : pip uninstall 패키지명      / conda remove 패키지명
