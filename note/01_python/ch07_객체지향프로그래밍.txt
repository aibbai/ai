7장 객체지향프로그래밍

1절. 객체와 클래스
	- class 클래스명 (클래스명은 항상 대문자로 시작)
	- 클래스도 독스트링 가능

2절. 생성자와 소멸자
	- 생성자(__init__(self, ...) : 객체가 생성될 때 자동 실행
		  		    : 인스턴스 변수 초기화 포함 -> 생성자 함수 중복정의(오버로딩) 불가
		                    : 없으면 default 생성자 자동 생성 
						* default 생성자 :  def __innit__(self)

									pass
						
					


	- 소멸자(__del__()) : 객체가 소멸될 때 자동 실행 (참조한 변수가 없어질 때)
				 : 소멸시 필요한 코드 포함
				 : 인스턴스 객체의 레퍼런스 카운트가 0이 될 때 실행

3절. 변수와 메소드

	- 클래스 변수 : 클래스 내부에 선언 => 생성된 모든 인스턴스(객체)에 동일한 값을 가짐
		   : 클래스 변수를 수정하면 모든 인스턴스에 반영될 수 O
		   : 객체 변수 수정하면 해당 객체에만 반영
